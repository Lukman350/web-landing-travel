# Web Travel Landing Page
Website Travel Landing Page dibuat dengan Next.js (TypeScript), Tailwind CSS

## Usage
Run Development
```bash
yarn dev
```
Run Production
```bash
# Build
yarn build

# Running
yarn start
```
